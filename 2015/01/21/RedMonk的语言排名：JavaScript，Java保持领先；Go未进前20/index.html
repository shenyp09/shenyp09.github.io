Red Monk的语言排名：Java Script，Java保持领先；Go未进前20

JavaScript在基于Github
和Stack Overflow
的排名统计中，均排名第一。

在RedMonk的这个月编程语言指数中：JavaScript，Java和PHP排名前三。这个指数基于Github
的语言代码行数和Stack Overflow上的语言标签数量。

“在狭隘的比较中，Java总是在排名中超过JavaScript，但二者的差异总是如此之远，以至于并不显著”，RedMonk首席分析师Stephen
O’Grady这样说。“最显著的特点是该语言经常被描述为将死，有时又被吹捧地上升几位”

RedMonk的指数中 Python, C#, C++, Ruby, CSS, C, 和
Objective-C也进入前十。排名11~20的语言为：Perl 和 Shell (并列11), R,
Scala, Haskell, Matlab, Go 和 Visual Basic (并列17), Clojure 和 Groovy
(并列19)。

在O’Grady认为有几种语言的排名指数还没有什么考查价值，包括Go, R 和
Switft。“在我们最近的指数轨迹分析中，Go进入前20还需要6~12个月的时间”，他说：“在半年以内，Go就可以超越Visual
Basic，Clojure和Groovy，取代CoffeeScript，进入第17名”。

R语言连续四个季度排在第13位。“即使R语言保持在13的位置，它仍然是最流行的统计语言。”，
O’Grady说，“虽然它面临着Python这样通用替代语言的竞争”。

O’Grady认为Swift，这个Apple的新语言是一个“神奇”的例子，它从上次的22位上升到了68位。“从列表中非常靠下的位置，上升到了CoffeeScript的后面和Lua的前面”。根据现在的统计数据，在今年的第三季度，它很可能进入前20。

CoffeeScript在出现的两年后跌出了前20.“这种编绎成JavaScript的语言将自己定位为既有JavaScript的流行，又避免了它的怪癖，但支持率似乎在逐渐下降”
，不过第三季度就可以看出来它是临时性的下降还是持续性的。

RedMonk指数是另外一个比较蓄积语言流行度的排名。每两年发表一次。它与
PYPL的编程语言排名（Pypl Popularity Programming Language
index）稍有不同：Java第一,PHP第二，JavaScript第7，PyPL将Google的搜索热度作为主要指标。另外一个排名：Tiobe，主要调查各主要网站，它的排名中C语言第一，Java第六，PHP第6，且JavaScript第7。但是JavaScript成为Tiobe
2014的年度语言。
